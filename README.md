## 五个问题
+ 软件工程对IT大学生的**意义**？
+ 软件工程的目标是协同开发高质量、高效率、凝集各种优点的软件系统，但其**高质量的定义**是什么，我们又**如何实现**？
+ 如何利用软件工程**分析**当下某些具体用户的**需求**？
+ 对于大型系统开发，**结构化开发方法**在软件开发中是否具有普适性，若不适用，**缺点**是？
+ **面向对象的开发方法**和**结构化开发方法**有何区别，分别有何用处？
